## Survival Prediction
The code for Survival Prediction tasks.

1. Organize your data into an excel file following the example `survival/splits/example.xlsx`
2. Specify the hyper-parameters in `survival/run.sh`
3. run by command
```bash
cd survival
bash run.sh
```
